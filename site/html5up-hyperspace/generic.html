  • Test Suite – is a collection of test cases that are grouped together to test a specific aspect of the software.

    Test Plan

  • Test Plan – is a description of the entire scope of testing work, description of the object, schedule and procedure, criteria for starting and ending testing, equipment required in the process of work, risk assessment and their solution

  • Test Scenario – is a document that describes a specific scenario in which the software will be used and the expected outcome.
